Woocommerce payment module for use with https://www.lamdaprocessing.com merchant accounts

== Description ==
NOTE: You must install the main "WooCommerce - excelling eCommerce" plugin in order to use this plugin

This module allows you to process payments using your https://www.lamdaprocessing.com merchant account

If you have problems with the CURL version, trying using the non-CURL option in the settings

== Installation ==

1. Unzip/Extract the plugin zip to your computer
2. Upload `\woocommerce-lamdaprocessing\` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to WP Admin -> Woocommerce -> Settings -> Payment Gateways -> Lamda Processing
5. Click the "Enable Lamda Processing" box, and enter your Merchant ID/API password etc.. for your merchant account
6. Save changes
